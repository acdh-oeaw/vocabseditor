The editor follows SKOS data model for the main elements of a controlled vocabulary. Dublin core schema is used to capture the metadata (such as date created, date modified, creator, contributor, source and other) about each element.

The user management system allows a user to share a created Concept Scheme with other users (called 'curators') to create new, edit and delete Concepts, Collections and Labels within this Concept Scheme. Each user can find all created elements and latest activity on user's page.
